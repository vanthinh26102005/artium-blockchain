## Scope

This inventory records the current backend runtime contract from repository evidence only: `src/main.ts`, `Dockerfile`, `.env.compose`, `docker-compose*.yml`, and package/runtime wiring. It is a current-state document for Phase 21 and intentionally stops before namespace, ingress, replica, or target Kubernetes decisions.
