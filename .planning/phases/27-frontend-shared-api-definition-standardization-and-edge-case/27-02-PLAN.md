<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User/domain-provided IDs -> URL path | IDs may contain characters that alter paths unless encoded. |
| Filter state -> query string | Numeric and boolean filter values must not be dropped or misencoded. |
| Environment config -> service routing | Artwork gateway/root/base variants must resolve without duplicating service prefixes. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-27-04 | Tampering | path params | mitigate | Encode all dynamic route segments with `encodePathSegment`. |
| T-27-05 | Denial of Service | query params | mitigate | Use one query helper that preserves valid `0` and `false` values and handles arrays predictably. |
| T-27-06 | Spoofing | artwork base URL routing | mitigate | Centralize artwork base compatibility and avoid accidental duplicate `/artwork` path routing. |
</threat_model>
