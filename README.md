Welcome to the **Artium Web Application** repository. This project is the final submission for the SE347 course (Web Application Development).

## 📂 Project Structure

This repository is organized into two main directories:

- **[BE](./BE)** (Backend): Contains the server-side code built with **NestJS**, following a microservices architecture.
- **[FE](./FE)** (Frontend): Contains the client-side code built with **Next.js 16**, **React 19**, and **Tailwind CSS**.

## 🚀 Getting Started

To get the full application running, you will need to set up and run both the backend and frontend services.

### Backend Setup

Please navigate to the `BE` directory and follow the instructions in [BE/README.md](./BE/README.md).

```bash
cd BE
# Refer to BE/README.md for installation and running instructions
# Typically involves: yarn install, docker setup, and yarn dev:all
```

### Frontend Setup

Please navigate to the `FE/artium-web` directory and follow the instructions in [FE/artium-web/README.md](./FE/artium-web/README.md).

```bash
cd FE/artium-web
# Refer to FE/artium-web/README.md for installation and running instructions
# Typically involves: npm install and npm run dev
```
