# Thông tin Kết nối Smart Contract

Tài liệu này chứa các thông tin cần thiết để Frontend hoặc Backend có thể kết nối và tương tác với Smart Contract `ArtAuctionEscrow` đang chạy trên Local Network.

## 1. Contract Address (Địa chỉ Contract)

Đây là địa chỉ duy nhất của Smart Contract sau khi deploy lên mạng.

> **Lưu ý:** Địa chỉ này **sẽ thay đổi** mỗi lần chúng ta chạy lệnh deploy mới hoặc khi khởi động lại mạng Local. Đây là địa chỉ mới nhất:
>
> ```text
> 0x5FbDB2315678afecb367f032d93F642f64180aa3
> ```

## 2. ABI (Application Binary Interface)

> path : smart-contracts/artifacts/contracts/ArtAuctionEscrow.sol/ArtAuctionEscrow.json

ABI là bản đồ mô tả toàn bộ các biến, các hàm (function) và sự kiện (event) có trong Smart Contract để Frontend biết cách gọi.

> **Lưu ý:** ABI **chỉ thay đổi** khi chúng ta chỉnh sửa code ở trong file `ArtAuctionEscrow.sol`. Nếu chỉ chạy deploy lại mà không sửa code thì ABI luôn giữ nguyên.
